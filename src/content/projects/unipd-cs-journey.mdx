## What this is

A public archive of my Computer Science degree at the University of Padua — built progressively as I go through the programme. For each course I've attended, the site collects the materials I produced: slides, exercise solutions, implementations, and any notes worth sharing.

The goal is straightforward: make everything accessible. If someone is taking the same courses, studying the same topics, or just curious about what a CS degree at UniPD looks like from the inside, they can follow along at their own pace.

## The site

Built as a Jekyll static site deployed on GitHub Pages, with Tailwind CSS for styling. Content is organised by year and semester, with a dedicated page per course.

The organisation and repository are currently private, pending a proper first release. Once the structure is solid and enough material is in place, everything goes public.

## Status

Active and growing. The private GitHub organisation (`stangacourses`) already contains material from the courses completed so far. Public release is planned as soon as the site structure is polished and the initial content is consistent across all covered courses.
