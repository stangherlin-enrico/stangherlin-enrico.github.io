## What this is

An open-source archive of my Computer Science degree at the University of Padua, built progressively as I move through the programme. For every course I attend, the site collects what I produce: lecture summaries, exercise solutions, exam strategy notes, and anything else worth sharing. The goal is simple — make it all findable and readable, for me and for anyone taking the same courses.

The project lives at [stangauni.github.io](https://stangauni.github.io) and is maintained under the [`StangaUni`](https://github.com/StangaUni) GitHub organisation. Content is licensed under CC BY-NC 4.0.

## The site

Built with React, TypeScript, and Vite, with MDX for content. Tailwind CSS handles styling; KaTeX renders LaTeX math inline and in display blocks; React Syntax Highlighter covers code. Deployed automatically to GitHub Pages via GitHub Actions.

Notes are organised by subject, each with its own changelog and a structured list of files grouped by type (summaries, exercises, extra material). A search bar on the home page filters by subject name; keyboard shortcut `Cmd/Ctrl+K` focuses it. Each note page gets an auto-generated table of contents and breadcrumb navigation.

## Courses covered so far

Three courses from year 1, semester 2 of the 2025/2026 academic year:

**Algebra e Matematica Discreta** — 12 CFU. Topics span complex numbers, divisibility, polynomials, modular arithmetic, congruence systems, matrices, vector spaces, recurrence relations, and combinatorics. 23 notes and several exercise sets.

**Programmazione** — 9 CFU. Fundamentals of imperative programming in C: algorithms, data types, control flow, functions, pointers, and arrays. 12 notes.

**Sistemi Operativi** — 9 CFU. Modern operating system principles: processes, threads, scheduling, synchronisation, deadlock, memory management, file systems, Unix/Linux and Windows internals. 9 notes.

## Status

Active and growing. 44 notes published so far across the three current courses, updated as lectures happen and material consolidates. Each course repository under `StangaUni` tracks changes in a dedicated changelog.
