## Overview

The site you're reading right now. Built from scratch as a personal space to document projects, write about software, and present a professional profile.

## Stack

- **React 18 + Vite 6** — fast dev server and optimised production build
- **TypeScript** — strict typing throughout
- **Tailwind CSS v3** — utility-first styling with the `@tailwindcss/typography` prose plugin for blog content
- **MDX** — blog posts and project pages written in Markdown with embedded React components
- **React Router v6** — client-side routing with a `404.html` fallback for GitHub Pages
- **Fuse.js** — lightweight client-side full-text search on blog posts
- **react-helmet-async** — per-page SEO meta tags
- **Puppeteer** — headless PDF generation for the Europass CV

## Features

- Blog with tag filtering, full-text search and MDX content
- Portfolio with per-project detail pages and multi-select tag filtering
- Europass CV with downloadable PDF (colour + black & white print-friendly versions)
- Dynamic download button driven by a `manifest.json` — add or remove CV versions without touching code
- GitHub Actions CI/CD — every push to `main` builds and deploys automatically
